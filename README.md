# Data Science & AI/ML Engineering Repository

Welcome! This repository serves as a centralized portfolio of my data science, machine learning, and deep learning implementations. The projects and assignments housed here focus on building foundational data pipelines, mastering feature engineering strategies, and deploying predictive frameworks.

Many of these implementations were developed and evaluated as part of my rigorous **Credit-Linked Program in Data Science & AI/ML via IIT Guwahati**.

* **Languages:** Python, SQL
* **Data Engineering & Visualization:** Pandas, NumPy, Matplotlib, Seaborn

---

## 📂 Repository Structure & Key Implementations

### 1. Advanced Feature Engineering & Data Preprocessing
* **One-Hot Encoding:** Implementing nominal categorical variable transformations for algorithms that require numeric distance metrics, avoiding arbitrary ordering.
* **Label Encoding:** Transforming target labels or binary categorical variables into discrete numeric formats efficiently.
* **Ordinal Encoding:** Mapping categorical features with an inherent, meaningful sequence or rank into structured numeric scales.
* **Data Pipelines:** Streamlining handling of missing data, feature scaling, outlier detection, and Exploratory Data Analysis (EDA).

### 2. Classical Machine Learning & Ensemble Methods
* **Random Forest & Decision Trees:** Implementing robust ensemble learning techniques for classification and regression tasks.
* **Model Evaluation:** Implementing hyperparameter tuning, confusion matrices, and ROC-AUC calculations to optimize prediction metrics.

### 3. Deep Learning Frameworks
* **Multilayer Perceptrons (MLP):** Building and tuning dense neural network architectures.
* **Deep Learning Workflows:** Utilizing TensorFlow for model training, optimization, and evaluation on real-world datasets.

## 🚀 Key Learning Outcomes
* **Feature Engineering Mastery:** Knowing exactly when to apply One-Hot vs. Label/Ordinal encoding to prevent target leakage and data sparsity.
* **End-to-End Pipelines:** Building pipelines that transition smoothly from raw, unstructured enterprise data to fully evaluated production models.
* **Software Engineering Integration:** Merging structured, clean Python backend design rules with data science scripts.
